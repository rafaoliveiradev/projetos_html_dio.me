Estrutura Básica

Antes de aprender as principais tags do HTML, é necessário conhecer um pouco de sua estrutura básica, que nada mais é do que algumas tags de configuração que irão receber as tags de elementos.

  • !DOCTYPE html

    • Primeira tag a ser colocada. É a tag que especifica que o tipo de documento é feito em HTML5

  • html

    • Tag raíz. Local onde todas as tags ficarão armazenadas

  • head

    • Configuração da página. Onde encontra-se metatags e tag de title por exemplo.

  • meta

    • Metatags, onde podemos configurar algo na página

  • title

    • Título da página web. Aparece na aba do navegador.

  • body

    • Corpo da página. Onde ficarão as tags de elementos, que cria os conteúdos visuais.

Tags de Cabeçalho

São tags utilizadas para inserir títulos e subtítulos.

  • h1 - Título de nível 1
  • h2 - Título de nível 2
  • h3 - Título de nível 3
  • h4 - Título de nível 4
  • h5 - Título de nível 5
  • h6 - Título de nível 6

Tag de Parágrafo

Cria-se um parágrafo para inserir textos simples através da tag p.

Tag de Links

Os links são úteis para que possamos navegar em mais páginas web. A tag a é a tag responsável pela criação de links, e dentro dela possui um atributo chamado href que é o local onde iremos inserir o caminho (link) da página para qual queremos navegar.

Tag de Imagens

Imagens são itens importantes em nosso site, talvez um dos itens muito utilizados em quaisquer aplicação web. Para inserir imagens, utilizamos a tag img. É importante citar que ao colocar a tag de imagem ela vem acompanhada de 2 atributos importantes:

  • src - source: Local onde a imagem está localizada.
  • alt - texto alternativo: Como se fosse uma legenda da imagem.

Além disso, podemos fazer com que as imagens possam ser manipuladas em relação a sua largura e altura . Para isso, podemos utilizar os atributos:

  • Width: Largura da imagem
  • Heigth: Altura da imagem

Obs: Ambas geralmente são medidas em px (pixels).

Tags de Formatações

Geralmente utilizamos algumas tags para fazer determinadas alterações em nossos textos. Para isso, temos aqui algumas tags que podem ajudar neste objetivo:

  • br - Quebra uma linha na página.
  • hr - Cria uma linha horizontal.
  • strong - Deixa o texto mais forte, em negrito.
  • em - Dar ênfase a alguma palavra/texto, ficando em itálico.
  • mark - Destaca algo com uma marca.
  • small - Texto fica um pouco menor.
  • del - Texto excluído. Passa um risco sobre o texto.
  • ins - Texto inserido. Conteúdo fica sublinhado.
  • sub - Texto fica um pouco mais abaixo.
  • sup - Texto fica um pouco mais acima.

Tags de Citações

Geralmente utilizados para inserir referências de algum conteúdo criado. Podendo ser referências de terceiros ou até mesmo do próprio desenvolvedor. As tags mais utilizadas:

  • blockquote - Usada para definir uma seção de outra fonte.
  • q - Tag para usar em citações curtas. O navegador insere aspas na citação.
  • abbr - Abreviar palavras / termos e mostrar seu significado ao passar o mouse.
  • address - Utilizadas para informações de contato. Número de telefone, e-mail, residência etc.
  • cite - Usado para definir um título de trabalho criativo. Poema, livro, música etc.
  • bdo - Muda o sentido do texto, geralmente deixando ao contrário. Para isso deverá utilizar o atributo dir="rtl".
